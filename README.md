# Gestor De Archivos usando Arbol General

Este proyecto simula un gestor de archivos similar al de Windows, permitiendo crear, modificar, eliminar, copiar y mover carpetas o archivos entre directorios. 

Su objetivo es aplicar los conocimientos adquiridos en la asignatura **Estructura de Datos II**, demostrando el uso de estructuras lineales (listas y arreglos), árboles generales para representar jerarquías y recursividad como técnica principal para la búsqueda y manipulación de nodos.

## Características
- Crear, modificar y eliminar carpetas o archivos.
- Copiar y mover archivos entre directorios.
- Representación jerárquica mediante un árbol general.
- Uso de recursividad para operaciones de búsqueda y gestión.
